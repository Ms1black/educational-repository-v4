# Information Processing Methods

Этот каталог содержит лабораторные работы по численным методам и аппроксимации.

- `lab1/` — интерполяция табличных функций (Лагранж, Ньютон, узлы Чебышёва, Рунге)

- `lab3/` — приближение табличных данных (МНК) + аппроксимация Паде
- `lab4/`, `lab5/`, `lab6/`, `lab7/` — заготовки под следующие лабораторные

## Как ориентироваться по файлам

- Основные решения оформлены в ноутбуках:

  - `lab2/main.ipynb`

## Лабораторная 1: Интерполяция табличных функций

**Тема:** сравнение интерполяционных полиномов Лагранжа и Ньютона, влияние типа узлов, феномен Рунге.

**Где смотреть:** `lab1/main.ipynb`

### Что реализовано

- классический Лагранж;
- барицентрическая форма Лагранжа;
- разделённые разности и форма Ньютона;
- узлы: случайные, равномерные, Чебышёва;
- сравнение ошибок и графики.

### Пример кода (идея)

```python
coef = divided_differences_table(x_nodes, y_nodes)
y_newton = newton_eval(x_nodes, coef, x_eval)
y_lagr = lagrange_classic(x_nodes, y_nodes, x_eval)
err = np.abs(y_true - y_lagr)
```

## Лабораторная 2: Сплайн-интерполяция

**Тема:** кубический сплайн, сравнение со полиномом, скорость сходимости.

**Где смотреть:** `lab2/main.ipynb` (и `lab2/main.py`)

### Что реализовано

- построение кубического сплайна с разными граничными условиями;
- для варианта используется `not-a-knot` (тип “с отсутствием узла”);
- сравнение `полином Ньютона vs сплайн` на функции `g(x)`;
- расчёт RMSE;
- log-log график зависимости RMSE от числа узлов;
- сравнение равномерных и Чебышёвских узлов.

### Пример кода (идея)

```python
spl = build_cubic_spline(x_nodes, y_nodes, bc_type="not-a-knot")
y_spline = spline_eval(spl, x_dense)
e = rmse(y_true, y_spline)
```

## Лабораторная 3: Приближение табличных функций

**Тема:** аппроксимация экспериментальных данных методом наименьших квадратов + аппроксимация Паде.

**Где смотреть:** `lab3/main.ipynb`

### Что реализовано

- МНК вручную в форме нормальных уравнений `A c = b`;
- модели для варианта 13:
  - квадратичная `{1, x, x^2}`,
  - полином 3-й степени `{1, x, x^2, x^3}`,
  - дробно-рациональная `a/(x+b)+c` (поиск `b` + линейный шаг по `a, c`);
- сравнение точности по СКО (RMSE);
- аппроксимация Паде `R_{3,2}` для `f(x)=x/(1+x^2)`.

### Пример кода (идея)

```python
B = np.column_stack([np.ones_like(x), x, x**2, x**3])
A = B.T @ B
b = B.T @ y
coef = np.linalg.solve(A, b)
y_hat = B @ coef
```
