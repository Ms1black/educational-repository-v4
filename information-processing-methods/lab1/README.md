## 2. Теоретическое введение

### 2.1. Постановка задачи интерполяции

Пусть на отрезке \([a, b]\) заданы **узлы интерполяции** (попарно различные):

\[
x_0, x_1, \ldots, x_n,\qquad x_i \in [a,b],
\]

и значения некоторой функции \(f(x)\) в этих точках:

\[
y_i = f(x_i),\quad i = 0,1,\ldots,n.
\]

**Задача интерполяции:** построить многочлен \(P_n(x)\) степени не выше \(n\), такой что

\[
P_n(x_i) = y_i,\quad i = 0,1,\ldots,n.
\]

Такой многочлен **существует и единственен** (при условии, что все узлы различны).

Интерполяционный многочлен можно записать в разных **эквивалентных формах** (Лагранжа, Ньютона и др.); они задают один и тот же многочлен, но по-разному удобны для вычислений и устойчивости.

---

### 2.2. Интерполяционный многочлен Лагранжа (классическая форма)

**Базисные многочлены Лагранжа:**

\[
\ell_i(x) = \prod_{\substack{j=0\\ j\neq i}}^{n} \frac{x - x_j}{x_i - x_j},\qquad i = 0,\ldots,n.
\]

Свойства: \(\ell_i(x_k) = \delta_{ik}\) (равен 1 в узле \(x_i\) и 0 в остальных).

**Многочлен Лагранжа:**

\[
L_n(x) = \sum_{i=0}^{n} y_i\,\ell_i(x).
\]

Очевидно, \(L_n(x_k) = y_k\).

**Вычислительная сложность (оценка):** при вычислении \(L_n(x)\) в одной точке \(x\) классическая форма требует порядка \(O(n^2)\) операций; при вычислении на \(m\) точках сетки — порядка \(O(m n^2)\).

---

### 2.3. Барицентрическая форма Лагранжа

Вводят **барицентрические веса** \(w_i\), не зависящие от \(x\):

\[
w_i = \frac{1}{\displaystyle\prod_{\substack{j=0\\ j\neq i}}^{n} (x_i - x_j)}.
\]

Тогда значение интерполяционного многочлена в точке \(x\) (если \(x\) не совпадает ни с одним узлом) записывают так:

\[
L_n(x) = \frac{\displaystyle\sum_{i=0}^{n} \frac{w_i\,y_i}{x - x_i}}{\displaystyle\sum_{i=0}^{n} \frac{w_i}{x - x_i}}.
\]

Если \(x = x_k\) для некоторого \(k\), по определению \(L_n(x_k) = y_k\) (в программе это обрабатывают отдельно).

**Замечание:** это тот же многочлен, что и в классической форме; барицентрическая запись часто **численно устойчивее** и удобнее для многократного вычисления на плотной сетке (веса считаются один раз).

---

### 2.4. Разделённые разности

**Нулевой порядок:** \(f[x_i] = y_i\).

**Разделённая разность порядка \(k\)** определяется рекурсивно:

\[
f[x_0,\ldots,x_k] = \frac{f[x_1,\ldots,x_k] - f[x_0,\ldots,x_{k-1}]}{x_k - x_0}.
\]

Коэффициенты \(f[x_0,\ldots,x_k]\) используются в форме Ньютона. Таблицу разностей удобно заполнять по столбцам (алгоритм «треугольник»).

---

### 2.5. Интерполяционный многочлен Ньютона

\[
N_n(x) = f[x_0] + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \cdots + f[x_0,\ldots,x_n]\prod_{j=0}^{n-1}(x-x_j).
\]

Снова \(N_n(x_i) = y_i\); при фиксированных узлах \(N_n(x) \equiv L_n(x)\) как многочлен.

**Добавление узла:** если появляется новый узел, к Ньютону можно добавить **один** новый член, не пересчитывая всё с нуля — практическое преимущество формы Ньютона.

**Сложность:** построение коэффициентов (разделённых разностей) — порядка \(O(n^2)\); вычисление значения в точке — порядка \(O(n)\) (схема Горнера по множителям \((x-x_j)\)).

---

### 2.6. Погрешность интерполяции

Пусть \(f \in C^{n+1}[a,b]\), узлы \(x_0,\ldots,x_n \in [a,b]\), \(P_n\) — интерполяционный многочлен степени \(\le n\). Тогда для любого \(x \in [a,b]\) существует \(\xi(x) \in [a,b]\) такое, что

\[
f(x) - P_n(x) = \frac{f^{(n+1)}(\xi(x))}{(n+1)!}\,\omega_{n+1}(x),
\qquad
\omega_{n+1}(x) = \prod_{j=0}^{n}(x-x_j).
\]

**Теоретическая оценка** максимальной погрешности на \([a,b]\):

\[
\max_{x\in[a,b]} |f(x)-P_n(x)| \le \frac{M_{n+1}}{(n+1)!}\,\max_{x\in[a,b]} |\omega_{n+1}(x)|,
\]

где \(M_{n+1} = \max_{t\in[a,b]} |f^{(n+1)}(t)|\) (если производная ограничена).

**Практическая оценка** на дискретной сетке точек \(\{z_k\}_{k=1}^{m}\):

\[
\varepsilon_{\max} = \max_k |f(z_k) - P_n(z_k)|,\qquad
\varepsilon_{\mathrm{mean}} = \frac{1}{m}\sum_{k=1}^{m} |f(z_k) - P_n(z_k)|.
\]

---

### 2.7. Выбор узлов: равномерная сетка и узлы Чебышёва

**Равномерные узлы** на \([a,b]\):

\[
x_i = a + i\frac{b-a}{n},\quad i=0,\ldots,n.
\]

**Узлы Чебышёва** (классические на \([-1,1]\), затем линейное отображение на \([a,b]\)). Например, узлы первого рода:

\[
\tilde{x}_k = \cos\frac{(2k-1)\pi}{2(n+1)},\quad k=1,\ldots,n+1 \quad \text{на } [-1,1],
\]

\[
x_k = \frac{a+b}{2} + \frac{b-a}{2}\,\tilde{x}_k.
\]

Узлы Чебышёва **сгущаются к концам** отрезка и уменьшают величину \(\max |\omega_{n+1}(x)|\) на \([a,b]\), что часто улучшает равномерную аппроксимацию.

**Случайные узлы** (равномерное распределение на \([a,b]\) с последующей сортировкой) используют для сравнения; при малых \(n\) результат может сильно зависеть от выборки — в отчёте целесообразно фиксировать **seed** генератора.

---

### 2.8. Феномен Рунге

Для некоторых гладких функций (классический пример: \(g(x)=\dfrac{1}{1+c x^2}\) на \([-1,1]\) при равномерных узлах) при увеличении \(n\) **максимальная ошибка на краях отрезка может расти** — это называют **феноменом Рунге**. Он связан с поведением \(\omega_{n+1}(x)\) при равномерной сетке.

**Смягчение:** переход на узлы Чебышёва (или другие «оптимальные» распределения), уменьшение степени, смена метода (сплайны и т.д. — за рамками данной лабораторной).

## 3. Описание используемых методов и алгоритмов

### 3.1. Общая схема эксперимента

1. Задать функцию \(f(x)\), отрезок \([a,b]\), число узлов \(n+1\) (или степень \(n\)).
2. Сформировать узлы: произвольные (случайные), равномерные, Чебышёва.
3. Вычислить \(y_i = f(x_i)\).
4. Построить многочлены **Лагранжа** (классический и барицентрический) и **Ньютона** (через разделённые разности).
5. На плотной сетке (например, 100–500 точек) вычислить приближённые значения и сравнить с \(f(x)\): графики, \(\varepsilon_{\max}\), \(\varepsilon_{\mathrm{mean}}\).
6. Замерить время построения/вычисления (например, `perf_counter`) при разных \(n\) для сравнения трудоёмкости реализаций.

### 3.2. Алгоритм Лагранжа (классический)

Для каждой точки \(x\) сетки:

- для каждого \(i\) собрать \(\ell_i(x)\) как произведение дробей \(\dfrac{x-x_j}{x_i-x_j}\);
- суммировать \(y_i \ell_i(x)\).

### 3.3. Алгоритм барицентрической формы

- Один раз вычислить веса \(w_i\) (произведения обратных разностей узлов).
- Для каждого \(x \notin \{x_i\}\) вычислить две суммы \(\sum w_i y_i/(x-x_i)\) и \(\sum w_i/(x-x_i)\) и взять отношение.
- Для \(x = x_k\) вернуть \(y_k\).

### 3.4. Алгоритм Ньютона

- Заполнить таблицу разделённых разностей по узлам \((x_i, y_i)\).
- Вычислить \(N_n(x)\) по схеме, аналогичной Горнеру: от старшего коэффициента к младшему с накоплением множителей \((x-x_j)\).

### 3.5. Сравнение методов в отчёте

Имеет смысл таблично сопоставить:

- погрешности для **случайных / равномерных / чебышёвских** узлов;
- **классический Лагранж / барицентрический Лагранж / Ньютон**;
- время выполнения при росте \(n\);
- для \(g(x)\) из варианта — иллюстрацию **Рунге** на равномерной сетке и его ослабление при узлах Чебышёва.
