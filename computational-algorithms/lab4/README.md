# Лабораторная работа 4

## Задача

Написать программу, сортирующую **главную диагональ** двумерного массива целых чисел методом **сортировки расчёской**.  
Матрица считывается из текстового файла:

- первая строка: количество строк `N` и столбцов `M`;
- следующие `N` строк: элементы матрицы.

## Формат входного файла

Пример (`input.txt`):

```text
4 5
9 2 3 4 5
6 1 8 7 3
5 4 0 6 2
8 9 1 2 4
```

## Файлы

- `index.ts` — основная программа (чтение файла + сортировка диагонали).
- `animation.ts` — консольная анимация шагов алгоритма.

## Запуск

```bash
npm start -- input.txt
```

Анимация:

```bash
npm run animate -- input.txt 400
```

Где:

- `input.txt` — путь к входному файлу;
- `400` — задержка между кадрами в мс (опционально, по умолчанию `500`).

## Блок-схема алгоритма

```mermaid
flowchart TD
    A([Старт]) --> B[Считать N, M и матрицу из файла]
    B --> C[diagLen = min(N, M)]
    C --> D[gap = diagLen; swapped = true]
    D --> E{gap > 1 ИЛИ swapped}
    E -- Нет --> Z([Конец: диагональ отсортирована])
    E -- Да --> F[gap = floor(gap / 1.247)]
    F --> G{gap < 1?}
    G -- Да --> H[gap = 1]
    G -- Нет --> I[оставить gap]
    H --> J[swapped = false; i = 0]
    I --> J
    J --> K{i + gap < diagLen?}
    K -- Нет --> E
    K -- Да --> L[j = i + gap]
    L --> M{A[i][i] > A[j][j]?}
    M -- Да --> N[Поменять A[i][i] и A[j][j]; swapped = true]
    M -- Нет --> O[Без обмена]
    N --> P[i = i + 1]
    O --> P
    P --> K
```

## Краткое описание анимации

В `animation.ts` визуализируется:

- текущий шаг алгоритма;
- текущее значение `gap`;
- сравниваемые элементы главной диагонали (жёлтая подсветка);
- элементы главной диагонали (cyan-подсветка);
- элементы в момент обмена (зелёная подсветка).

## Блок-схема анимации (`animation.ts`)

```mermaid
flowchart TD
    A([Старт]) --> B[Считать путь к файлу и задержку]
    B --> C[Прочитать матрицу из файла]
    C --> D[diagLen = min(N, M)]
    D --> E{diagLen <= 1?}
    E -- Да --> F[Показать один кадр<br/>"Недостаточно элементов"]
    F --> Z([Конец])
    E -- Нет --> G[gap = diagLen; swapped = true; step = 0]
    G --> H[Показать стартовый кадр]
    H --> I{gap > 1 ИЛИ swapped}
    I -- Нет --> Y[Показать финальный кадр<br/>"Диагональ отсортирована"]
    Y --> Z
    I -- Да --> J[gap = floor(gap / 1.247); если gap < 1, то gap = 1]
    J --> K[swapped = false; i = 0]
    K --> L{i + gap < diagLen?}
    L -- Нет --> I
    L -- Да --> M[j = i + gap; step = step + 1]
    M --> N[Показать кадр сравнения]
    N --> O{A[i][i] > A[j][j]?}
    O -- Да --> P[Обменять A[i][i] и A[j][j]; swapped = true]
    P --> Q[Показать кадр обмена]
    O -- Нет --> R[Показать кадр<br/>"Обмен не нужен"]
    Q --> S[i = i + 1]
    R --> S
    S --> L
```
